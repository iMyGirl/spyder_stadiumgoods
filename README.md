## 说明：

0. result.md中记录了colab平台上的运行结果，可参考；
1. 在主程序中添加 url 即可运行 code.py 文件；
2. 运行 code.py 文件生成的csv文件为分页爬取结果；
3. 可通过其他脚本将分页爬取结果汇总至一个文件中;
4. 主要通过BeautifulSoup数据包单线程爬取，分析url指向页面，伪装header爬取。
